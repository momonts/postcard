# postcard

Reddit like dApp

Subreddits == Stamps Redditors == mailers

Differences:

Currency:

each comments you can earn "penny";

"penny" is a currency for Postcard;

You can use "penny" for creating forum;

customizing profile.

0.0001 sol == 1 penny

Costs/interaction:

upvotes & downvotes (0.002 sol)
comments (0.0001 sol)
posts (penny or 0.0003 sol)
Colors:

lime-500
